# My personal dotfiles
My personal dotfiles using [iTerm2](https://iterm2.com/)
 with [oh-my-zsh](https://ohmyz.sh/).

## Required
- macOS

## Install
1. Download installation materials.
    ```bash
    git clone https://github.com/Tom-index/dotfiles.git ~/dotfiles
    ```
2. Setup
    ```bash
    cd ~/dotfiles && make
    ```
